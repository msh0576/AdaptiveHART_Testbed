Interface: AsyncReceive

interface AsyncReceive

Provides packet reception events in an asynchronous fashion.

Author:
Greg Hackmann
See:
Receive

Commands
command void updateBuffer(message_t *msg) Provides a new message buffer to the radio stack.

Events
event void receive(message_t *msg, void *payload, uint8_t len) A packet was received over the radio.

Commands - Details

updateBuffer

command void updateBuffer(message_t *msg)

Provides a new message buffer to the radio stack. This should be called (only) after each packet is handled, since the radio stack may do additional radio-specific event handling here.

Parameters:
'message_t * ONE msg' the new message buffer to provide to the radio stack (possibly the same one passed up from the receive event)

Events - Details

receive

event void receive(message_t *msg, void *payload, uint8_t len)

A packet was received over the radio. After it is processed, updateBuffer should be called.

Parameters:
'message_t * ONE msg' the packet that was received
'void * COUNT(len) payload' the packet's payload
len - the packet's length
