Interface: AsyncSend

interface AsyncSend

Provides packet sending commands in an asynchronous fashion.

Author:
Greg Hackmann
See:
AMSend

Commands
command error_t cancel(message_t *msg) Attempts to cancel a packet transmission.
command void *getPayload(message_t *msg, uint8_t len) Gets the packet payload.
command uint8_t maxPayloadLength() Gets the maximum packet payload length.
command error_t send(message_t *msg, uint8_t len) Sends a packet over the radio, in split-control fashion.

Events
event void sendDone(message_t *msg, error_t error) The packet sending operation has completed.

Commands - Details

cancel

command error_t cancel(message_t *msg)

Attempts to cancel a packet transmission.

Parameters:
'message_t * ONE msg' the packet to cancel
Returns:
whether the cancellation succeeded

getPayload

command void *getPayload(message_t *msg, uint8_t len)

Gets the packet payload.

Parameters:
'message_t * ONE msg' the packet
Returns:
'void * COUNT_NOK(len)' the packet's payload

maxPayloadLength

command uint8_t maxPayloadLength()

Gets the maximum packet payload length.

Returns:
the maximum packet payload length, in bytes

send

command error_t send(message_t *msg, uint8_t len)

Sends a packet over the radio, in split-control fashion.

Parameters:
'message_t * ONE msg' the packet to send
len - the packet's length
Returns:
SUCCESS if the packet was buffered; EOFF if the radio is off; or FAIL if the request could not otherwise be completed

Events - Details

sendDone

event void sendDone(message_t *msg, error_t error)

The packet sending operation has completed.

Parameters:
'message_t * ONE msg' the packet that has been completely processed
error - SUCCESS if the packet was successfully sent over the radio, or FAIL otherwise
